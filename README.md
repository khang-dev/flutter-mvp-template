## Key Features

* MVP Core boilerplate code
* Localization (l10n)
* Flavors: dev + prod
* App Icons generator

## How To Use

Create new Flutter project repository with this template repository

From your root project:

```bash
# Get all packages
$ ./helper get

# Reload the project
$ ./helper reload

# Generate App Icon (Android + iOS) from the folder assets/dev_assets/
$ ./helper genAppIcon
```
